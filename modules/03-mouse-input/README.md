# 03-mouse-input

마우스 입력과 좌표를 확인하는 모듈입니다.

이 모듈에서는 Windows 콘솔 환경에서 마우스 좌표와 마우스 버튼 상태를 읽는 가장 기본적인 흐름을 연습합니다.

## What This Module Covers

- `POINT` 구조체로 좌표 다루기
- `GetCursorPos()`로 현재 마우스 위치 읽기
- `GetAsyncKeyState()`로 좌클릭과 우클릭 상태 확인하기
- `_kbhit()`와 `_getch()`로 종료 키 받기
- `Sleep()`으로 반복 속도 조절하기

## Current Example

현재 예제는 다음 흐름을 연습합니다.

1. 반복문 안에서 현재 마우스 좌표를 읽는다.
2. 좌클릭이 눌려 있는지 확인한다.
3. 우클릭이 눌려 있는지 확인한다.
4. 현재 좌표와 버튼 상태를 콘솔에 출력한다.
5. `q` 또는 `Q`를 누르면 프로그램을 종료한다.

## Learning Point

이 모듈의 핵심은 "마우스 입력은 위치 정보와 버튼 상태를 함께 다뤄야 한다"는 점을 이해하는 것입니다.

또한 이 예제를 통해 다음 개념을 함께 익힐 수 있습니다.

- `POINT`는 x, y 좌표를 저장하는 구조체이다.
- `GetCursorPos()`는 현재 마우스 좌표를 읽어온다.
- `GetAsyncKeyState()`는 특정 버튼이 현재 눌려 있는지 확인한다.
- `Sleep()` 값이 커질수록 화면 갱신 속도는 느려진다.

## Next Extension

- 특정 영역 안에서 클릭 판정하기
- 더블 클릭 감지하기
- 마우스 휠 입력 확인하기
