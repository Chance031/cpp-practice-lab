# 03-mouse-input

마우스 입력과 좌표를 확인하는 모듈입니다.

이 모듈에서는 Windows 콘솔 환경에서 마우스 위치를 읽고, 좌클릭과 우클릭이 어떻게 감지되는지 확인하는 가장 기본적인 흐름을 연습합니다.

## What This Module Covers

- Windows API를 이용한 마우스 위치 읽기
- 좌클릭과 우클릭 입력 감지
- 반복문으로 마우스 상태 계속 확인하기
- `q` 입력으로 프로그램 종료하기

## Current Example

현재 예제는 다음 흐름을 연습합니다.

1. 현재 마우스 좌표를 계속 확인한다.
2. 좌클릭이 눌렸는지 확인한다.
3. 우클릭이 눌렸는지 확인한다.
4. `q` 또는 `Q`를 누르면 프로그램을 종료한다.

## Learning Point

이 모듈의 핵심은 "마우스 입력은 키보드 입력과 다르게 위치 정보와 버튼 상태를 함께 다뤄야 한다"는 점을 이해하는 것입니다.

즉, 단순히 버튼 하나를 읽는 것을 넘어서, 화면 좌표와 입력 상태를 함께 확인하는 첫 단계라고 볼 수 있습니다.

## Next Extension

이 모듈을 더 확장하고 싶다면 다음 정도를 시도해볼 수 있습니다.

- 좌표가 바뀔 때만 출력하기
- 더블 클릭 감지하기
- 마우스 휠 입력 확인하기
- 특정 좌표 범위 안에서 클릭 판정하기
