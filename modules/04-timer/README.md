# 04-timer

시간의 흐름을 확인하고, 시작 후 얼마나 지났는지 계산하는 기초를 연습하는 모듈입니다.

이 모듈에서는 게임에서 자주 쓰이는 타이머의 가장 기본적인 개념인 "시작 시각과 현재 시각의 차이"를 이용한 시간 계산을 다룹니다.

## What This Module Covers

- `std::chrono::steady_clock`으로 현재 시각 가져오기
- 시작 시각과 현재 시각의 차이 계산하기
- `duration_cast`로 시간 단위 변환하기
- `std::chrono::seconds`로 초 단위 다루기
- `std::flush`로 같은 줄 출력 바로 갱신하기
- `sleep_for()`로 반복 속도 조절하기

## Current Example

현재 예제는 다음 흐름을 연습합니다.

1. 프로그램이 시작된 시각을 저장한다.
2. 반복문 안에서 현재 시각을 계속 확인한다.
3. 시작 후 몇 초가 지났는지 계산한다.
4. 지난 시간을 같은 줄에 계속 출력한다.
5. 5초가 지나면 프로그램을 종료한다.

## Learning Point

이 모듈의 핵심은 "시간은 직접 세는 것이 아니라, 시작 시각과 현재 시각의 차이로 계산한다"는 점을 이해하는 것입니다.

또한 이 예제를 통해 다음 개념을 함께 익힐 수 있습니다.

- `steady_clock`은 일정하게 흐르는 시계를 제공한다.
- `duration_cast`는 시간 차이를 원하는 단위로 변환한다.
- `std::chrono::seconds`는 초 단위를 뜻한다.
- `std::flush`는 출력 버퍼의 내용을 바로 화면에 반영한다.
- `sleep_for()` 값이 커질수록 화면 갱신 속도는 느려진다.
