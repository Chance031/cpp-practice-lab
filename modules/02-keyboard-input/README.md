# 02-keyboard-input

키보드 입력을 받고, 입력에 따라 상태가 어떻게 바뀌는지 확인하는 모듈입니다.

이 모듈에서는 실시간 입력보다 먼저, 가장 단순한 콘솔 입력으로 "키를 받는다 -> 값이 바뀐다 -> 출력으로 확인한다" 흐름을 익히는 것을 목표로 합니다.

## What This Module Covers

- `std::cin`을 이용한 기본 키 입력
- 입력값에 따라 좌표 변경하기
- 잘못된 입력에 반응하기
- 반복문으로 입력을 계속 받기
- `q` 입력으로 종료하기

## Current Example

현재 예제는 다음 흐름을 연습합니다.

1. 플레이어의 현재 위치를 출력한다.
2. `w`, `a`, `s`, `d` 중 하나를 입력받는다.
3. 입력에 따라 좌표를 이동시킨다.
4. 이동 결과를 다시 출력한다.
5. `q`를 입력하면 프로그램을 종료한다.

## Learning Point

이 모듈의 핵심은 "입력을 받으면 상태가 바뀌고, 그 결과를 출력으로 확인한다"는 흐름을 익히는 것입니다.

즉, `output` 모듈에서 만든 출력 감각을 실제 입력 처리와 연결하는 첫 단계라고 볼 수 있습니다.

## Next Extension

이 모듈을 더 확장하고 싶다면 다음 정도를 시도해볼 수 있습니다.

- 대문자 입력도 처리하기
- 플레이어 상태(`Idle`, `Move`) 추가하기
- 잘못된 입력 횟수 세기
- 입력 버퍼와 `std::getline` 차이 이해하기
