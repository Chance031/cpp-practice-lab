# 02-keyboard-input

키보드 입력을 받고, 일반 키와 특수 키가 어떻게 들어오는지 확인하는 모듈입니다.

이 모듈에서는 `Enter`를 누르지 않아도 입력을 바로 읽을 수 있는 `_getch()`를 사용해, 일반 키와 방향키를 구분해서 확인하는 흐름을 연습합니다.

## What This Module Covers

- `_getch()`를 이용한 즉시 키 입력
- 일반 키의 문자와 코드 값 출력
- 특수 키 입력 방식 이해하기
- 방향키 코드를 이름으로 구분하기
- `q`, `Q` 입력으로 종료하기

## Current Example

현재 예제는 다음 흐름을 연습합니다.

1. `_getch()`로 키 입력을 즉시 받는다.
2. 입력이 일반 키인지 특수 키인지 구분한다.
3. 일반 키는 문자와 코드 값을 출력한다.
4. 방향키는 이름과 코드 값을 출력한다.
5. `q` 또는 `Q`를 입력하면 프로그램을 종료한다.

## Learning Point

이 모듈의 핵심은 "키보드 입력이 Enter 없이 바로 들어올 수 있다"는 점과, "특수 키는 일반 키와 다른 방식으로 처리해야 한다"는 점을 이해하는 것입니다.

즉, 단순히 문자를 입력받는 것을 넘어서, 키보드 입력 구조 자체를 확인하는 단계라고 볼 수 있습니다.

## Next Extension

이 모듈을 더 확장하고 싶다면 다음 정도를 시도해볼 수 있습니다.

- `ESC` 키도 종료 키로 처리하기
- 숫자 키와 문자 키의 코드 값 비교하기
- 입력한 방향키에 따라 좌표 이동시키기
- 플레이어 상태와 입력을 연결하기
