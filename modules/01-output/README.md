# 01-output

출력 기능을 연습하는 모듈입니다.

이 모듈에서는 단순히 문자열을 출력하는 데서 끝나지 않고, 게임에서 자주 보게 되는 상태 정보를 콘솔에 보여주는 흐름을 연습합니다.

## What This Module Covers

- `std::cout`를 이용한 기본 출력
- 함수로 출력 로직 분리하기
- HP, MP, Level, EXP 같은 상태값 출력
- 상태이상에 따른 수치 변화 출력
- `enum class`를 이용한 상태 관리
- 경험치 획득과 레벨업 결과 출력

## Current Example

현재 예제는 다음 흐름을 콘솔에서 확인할 수 있습니다.

1. 플레이어의 초기 상태 출력
2. 몬스터의 공격으로 HP 감소
3. `Burn` 상태이상 적용
4. 상태이상으로 인한 추가 데미지 적용
5. 포션 사용으로 HP 회복 및 상태 회복
6. 스킬 사용으로 MP 감소
7. 몬스터 처치 후 경험치 획득
8. 경험치가 최대치를 넘기면 레벨업 처리

## Learning Point

이 모듈의 핵심은 "출력은 단순한 문장 표시가 아니라, 게임 상태를 확인하는 도구"라는 점입니다.

즉, 값을 바꾸고 그 결과를 출력으로 확인하는 흐름을 익히는 것이 목표입니다.

## Next Extension

이 모듈을 더 확장하고 싶다면 다음 정도를 시도해볼 수 있습니다.

- `Poison` 같은 상태이상 추가
- MP 회복 예시 추가
- 여러 번 레벨업하는 경험치 처리
- 상태창과 이벤트 로그 출력 형식 더 다듬기

하지만 현재 예제만으로도 `output` 모듈의 목적에는 충분히 잘 맞습니다.
