# cpp-practice-lab

문서 정리와 README 작성에는 AI의 도움을 함께 활용합니다.

게임을 만들기 전에 필요한 기능들을 작은 실험 단위로 나누어 연습하는 C++ 학습용 저장소입니다.

예를 들면 다음과 같은 주제를 각각 독립적인 모듈처럼 다룹니다.

- 출력
- 키보드 입력
- 마우스 입력
- 시간 처리
- 충돌 판정
- 간단한 렌더링

각 폴더는 "하나의 기능을 이해하고 실험하는 작은 프로젝트"라는 기준으로 관리합니다.

## Goal

큰 게임 프로젝트에 바로 들어가기 전에, 자주 쓰게 될 기능을 작게 나누어 직접 만들어 보고 이해합니다.

이 저장소에서는 다음 흐름을 목표로 합니다.

1. 기능 하나를 정한다.
2. 가장 작은 예제로 동작을 확인한다.
3. 입력과 출력을 눈으로 확인한다.
4. 나중에 게임 프로젝트에서 재사용할 수 있게 정리한다.

## Structure

```text
modules/
  01-output/
  02-keyboard-input/
  03-mouse-input/
```

앞으로 주제가 늘어나면 같은 규칙으로 추가하면 됩니다.

예시:

- `04-timer`
- `05-collision`
- `06-scene-state`

## Module Rules

각 모듈은 가능하면 아래 내용을 포함합니다.

- 무엇을 배우는지
- 어떤 입력을 받는지
- 어떤 결과를 확인할 수 있는지
- 다음 단계에서 무엇을 확장할 수 있는지

## Suggested Workflow

모듈을 만들 때는 아래 방식으로 진행하면 관리가 편합니다.

1. 한 번에 하나의 기능만 다룬다.
2. 예제는 가능한 한 작게 만든다.
3. 바로 실행해서 눈으로 확인할 수 있게 만든다.
4. 배운 점은 해당 모듈 폴더에 간단히 기록한다.
